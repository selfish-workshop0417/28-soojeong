# 씬스틸러 (SceneStealer) — PRD

## 1. 서비스 개요
씬스틸러는 영상 창작자들이 자신에게 영감을 준 레퍼런스 영상을 "왜 훔쳐왔는지"와 함께 기록하고 공유하는 큐레이션 커뮤니티다. 유튜브, 인스타그램, Vimeo 등 플랫폼에 흩어진 레퍼런스를 한 곳에 모으고, 다른 창작자의 취향을 엿보며 영감을 주고받는다. 개인의 아카이빙 도구와 창작자 네트워크를 결합한 서비스다.

## 2. 폴더 구조
이 레포는 5개 팀(파트)에 따른 5개 영역으로 나뉜다:
- planning/ — 기획 (무엇을 만들 것인가)
- design/ — 디자인 (어떻게 보여줄 것인가)
- cx/ — CX (누가 어떻게 사용할 것인가)
- marketing/ — 마케팅 (어떻게 알릴 것인가)
- strategy/ — 전략 (어떻게 지속 가능하게 만들 것인가)

각 영역의 상세는 해당 폴더에서 작성된다.

## 3. 기획 (Planning)
핵심 기능:
- 여러 플랫폼 영상 링크 저장 및 썸네일 자동 추출
- 영상별 "훔친 이유" 코멘트 작성
- 태그·카테고리 분류 (무드, 장르, 기법 등)
- 컬렉션(무드보드) 생성 및 공유
- 다른 창작자 아카이브 탐색·팔로우

MVP 범위: 링크 저장 + 이유 기록 + 개인 아카이브 공개 페이지.

상세는 `planning/` 폴더 참조. [작성 예정]

## 4. 디자인 (Design)
무드보드처럼 영상 썸네일이 큼직하게 흐르는 비주얼 중심 레이아웃. 창작자의 개성이 드러나는 "훔친 장면 앨범" 감성으로, 감각적이고 덜 정돈된 아트 디렉션을 지향한다.
주요 화면 흐름: 피드 탐색 → 창작자 아카이브 → 영상 상세(이유 보기) → 내 컬렉션에 저장.

상세는 `design/` 폴더 참조. [작성 예정]

## 5. CX (Customer Experience)
타겟 페르소나: "남의 작업물에서 영감을 얻지만, 기존 북마크·메모앱으로는 정리가 안 되는 20~30대 영상 창작자".
사용자 여정 핵심 단계: 발견(영감 영상 마주침) → 저장(링크+이유 기록) → 탐색(타인의 아카이브 훔쳐보기) → 재창작(내 작업에 반영).

상세는 `cx/` 폴더 참조. [작성 예정]

## 6. 마케팅 (Marketing)
첫 사용자는 영상 편집·크리에이터 커뮤니티(디스코드·오픈채팅·트위터) 시드 멤버 중심으로 확보. "당신의 레퍼런스 서랍, 공개해볼래요?" 같은 초대형 메시지로 진입.
주요 채널: 인스타그램 릴스(서비스 자체 콘텐츠화) + 창작자 커뮤니티 직접 홍보.

상세는 `marketing/` 폴더 참조. [작성 예정]

## 7. 전략 (Strategy)
초기에는 무료 개방으로 사용자·콘텐츠를 축적하고, 이후 프리미엄 컬렉션·협업 기능·스튜디오 단위 팀 플랜으로 수익화를 검토한다.
성장 방향: 개인 아카이빙 도구 → 창작자 네트워크 → 영상 레퍼런스 데이터베이스.

상세는 `strategy/` 폴더 참조. [작성 예정]
