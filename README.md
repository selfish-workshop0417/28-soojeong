# 씬스틸러 (SceneStealer)

영상 창작자들이 각자의 레퍼런스 영상을 이유와 함께 모으고, 서로의 아카이브를 훔쳐보는 큐레이션 커뮤니티.

## 프로젝트 정보
- **유형**: 커뮤니티 + 콘텐츠 아카이브 (영상 레퍼런스 큐레이션)
- **타겟 사용자**: 유튜버, 릴스/쇼츠 크리에이터, 영상 편집자, 감독 등 영상 창작자
- **출시 목표**: [작성 예정]

## 핵심 기능
- 모든 플랫폼(YouTube, Instagram, Vimeo 등) 영상 링크 저장
- 각 영상에 "왜 이 장면을 훔쳤는지" 이유/코멘트 기록
- 태그/카테고리로 분류 (장르, 무드, 기법 등)
- 다른 창작자의 아카이브 탐색·팔로우
- 컬렉션(무드보드) 단위로 큐레이션·공유

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
